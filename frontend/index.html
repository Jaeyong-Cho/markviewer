Welcome

Welcome to MarkViewer

A powerful markdown viewer with PlantUML, Mermaid, and search capabilities.

Features

  • Render markdown with JavaScript
  • PlantUML diagrams using local plantuml.jar (SVG format)
  • Mermaid diagram rendering
  • Recursive directory sidebar navigation
  • Full-text search functionality
  • GitHub-style code highlighting
  • Tabbed interface for multiple files
  • Split screen for side-by-side viewing

Getting Started

  1. Enter a workspace directory path in the input field at the top of the sidebar
  2. Press Enter to load the directory and browse files
  3. Click on files to open them in tabs
  4. Use the search box to find content across all files
  5. View rendered markdown with diagrams and highlighted code

Tab Features

  • Open multiple files in tabs
  • Click tabs to switch between files
  • Close tabs with the × button or middle-click
  • Keyboard shortcuts: Ctrl+W to close active tab

Split Screen

  • Toggle split screen mode with Ctrl+Shift+\
  • Drag tabs between left and right panes
  • Independent file browsing in each pane
